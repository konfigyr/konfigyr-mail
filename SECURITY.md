# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| 1.0.x | ✅ |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Use GitHub's private vulnerability reporting instead:

1. Go to <https://github.com/konfigyr/konfigyr-mail/security/advisories/new>
2. Fill in the description, affected versions, and steps to reproduce.
3. Submit the report.

You will receive an acknowledgement within 72 hours. We aim to release a fix within 14 days of confirmation, depending on severity and complexity.
